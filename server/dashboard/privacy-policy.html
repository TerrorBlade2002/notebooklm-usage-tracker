NotebookLM Usage Tracker Privacy Policy

Last updated: April 4, 2026

NotebookLM Usage Tracker is used to record NotebookLM usage activity for internal analytics, reporting, operational tracking, and audit purposes.

Information collected

The extension and server may collect and process the following information:

  • Windows system username of the signed-in agent
  • Notebook name
  • Notebook ID
  • Session ID
  • Turn number
  • First question summary
  • Current question summary
  • Timestamps related to activity

How the information is used

This information is used only for the following purposes:

  • Monitoring NotebookLM usage across agents
  • Generating operational dashboards and reports
  • Supporting audit, compliance, and internal review processes
  • Analyzing adoption and usage trends

How data is transmitted

Collected data is transmitted from the browser extension to the organization’s configured tracking server endpoint over HTTPS where available.

How data is stored

Data is stored on the organization’s server infrastructure and retained according to the organization’s internal policies and operational requirements.

Data sharing

This extension is intended for internal organizational use. Data collected by the extension is not sold. Data may be accessed by authorized internal administrators, IT teams, operations teams, compliance teams, or management personnel as required for business operations.

User choices

Use of this extension is controlled by the deploying organization. If you are an end user and have questions about your data, retention, or access, contact your internal IT or compliance team.

Security

Reasonable administrative and technical measures should be used by the deploying organization to protect collected information. However, no method of transmission or storage is guaranteed to be completely secure.

Contact

For questions about this deployment, contact the organization or IT team responsible for operating this extension and server.

Policy updates

This privacy policy may be updated from time to time. Any updates will be posted at this URL.
